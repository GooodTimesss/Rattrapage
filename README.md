# Cybersecure_projet
Bienvenue sur le dépôt GitHub associé au projet Cybersecure.

## Informations importantes

Ce dépôt **n'est pas destiné à être utilisé pour l'installation principale** de l'infrastructure. Toutes les machines virtuelles ont été préconfigurées, testées, et sont disponibles via un lien de téléchargement déposé sur Monday.

## À quoi sert ce dépôt GitHub ?

Ce dépôt est fourni uniquement :

- pour effectuer certaines configurations manuelles ponctuelles, si nécessaire ;
- en **dernier recours**, si un bug empêchait l’utilisation des VMs préconfigurées (cas peu probable car elles ont été testées sur plusieurs machines) ;
- pour **illustrer notre méthode de travail** en équipe durant le projet Cybersecure (suivi des fichiers, versionnement, étapes intermédiaires...).

## Contenu du dépôt

- Fichiers XML d’export de la configuration pfSense
- Fichiers de configuration manuelle
- README explicatifs des étapes de configuration
- Extraits de commandes utilisées

## Recommandation

Utilisez ce dépôt uniquement si une configuration manuelle s’impose. Dans tous les autres cas, privilégiez l’importation des VMs fournies, qui contiennent l’intégralité de l’infrastructure prête à l’emploi.

---

Pour toute question ou besoin d’assistance, veuillez vous référer au manuel d'installation fourni avec le projet.
